## Common Issues

**"Allocations don't match!"**
- Make sure your allocated amounts equal the total loot
- Use the "Split Evenly" button for quick fix

**"Insufficient funds!"**
- Character doesn't have enough in their account
- Try deducting from Party Vault instead
- Or record less amount

**"Can't find a character account?"**
- Make sure player has set a "current character"
- Go to Players page and set it
- Then come back and try again

**"Where's my spending history?"**
- Go to "ğŸ’¸ Spending Log" tab
- Use filters if looking for specific account/category
- Most recent appears first

## Pro Tips for Game Masters

1. **End of Session Routine**:
   - Add all loot discovered that session
   - Link to session number for easy reference
   - Record all party spending
   - Takes 5-10 minutes total

2. **Before Session Starts**:
   - Quick glance at wallets to know party wealth
   - Check spending log for trends (are they broke?)
   - Reference loot log for what they've found

3. **Financial Disputes**:
   - Click a character's name
   - Show them their exact balance from Wallets tab
   - Point to specific loot/spending in logs

4. **Campaign Milestones**:
   - "First 100 gp!" - celebrate in Wallets
   - "We've spent HOW MUCH on taverns?" - check Spending Log filtered by Food
   - "Total plunder" - sum of Party Vault + all Character Accounts

5. **Patron Campaigns**:
   - Set patron percentage in Settings
   - Auto-allocation takes care of cuts automatically
   - See patron balance in Wallets tab
